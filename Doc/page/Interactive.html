Form

E una parte del documento che contiene controlli di input. Gli attributi obbligatori sono action (specifica url) e method (metodo HTTP get o post). Get usato per i dati in chairo mentre il post usato come pseudo-riservati (serve anche https).

<form action="wwww.google.it" methos="post"> </form>

Label

Ogni controllo deve avere una label che descriva il controllo dell'utente. E fondamentale per l'accessibilità. Per relazionare la label si puo usare anche l'attributo id.

<form action="wwww.google.it" methos="post"> <label for="CN"> Customer name: </label> <input id="CN" /> </form>

Fieldset

Elemento che raggruppa puù controlli che hanno semantica comune.

<form action="wwww.google.it" methos="post"> <fieldset> <label for="CN"> Customer name: </label> <input id="CN" /> <label for="Cc"> Customer cognome: </label> <input id="Cc" /> </fieldset> <input type="submit"/> </form>

Input

Consente di inserire nella pagina molti tipi diversi di controllo. Il tipo puo essere specificato con l'attributo type (text, search, submit, button, image ...). Nel caso del type uguale a button puo avere l'attributo onclick per associarli uno script javascript.

<input type="submit" value="text" />

Textarea

Definisce un controllo di input per testi multimediali. Si possono specificare gli attributi cols e rows per le dimensioni.

<form action="wwww.google.it" methos="post"> <textarea cols="4" rows="5"> Qui il testo da inviare </textarea> </form>

Select

Usato per creare menu a tendina. Le diverse opzioni sono introdotte dal tag option.

<Select name="tendina"> </Select>

Option

Introduce le opzioni del tag select. Si puo usare il tag optgroup per raggruppare piu option di una stessa famiglia. per dare una struttura piu chiara e accessibile.

<select name="sport"> <option> Calcio </option> <option> Tennis </option> <option> Golf </option> </select>

Output

Rappresenta il risultato di un calcolo o azione dell'utente.

<form oninput="x.value=parseInt(a.value)"> <label for="a"> slider </label> <input type="range" id="a" /> <output name="x" for="a"> </output> </form>
