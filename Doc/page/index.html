Struttura di tag obbligatori per creare un documento HTML

<!DOCTYPE html> <html> <head> <title> Nome </title> <meta charset="UTF-8" /> </head> <body> Content of the document </body> </html>

Metadati

Gli elementi di metadazione consentono di descrivere il documento. Spesso usati dai motori di ricerca (name="description" usato per lo sniplet di google).

I metadati sono inclusi nell'head del documento.

  • <title> </title>
  • <base> </base>
  • <link> </link>
  • <meta> </meta>
  • <style> </style>

Title

Rappresenta il titolo o il nome del documento (della scheda). Ogni documento ne deve avere al massimo 1.

<head> <title> Nome della scheda </title> </head>

Base

  • Questo elemento dev'essere unico all'interno del documento, se vi sono pi√π elemento base, viene preso in considerazione solo il primo.
  • Lo scopo del base e quella di indicare il path base del documento che serivra per risolvere gli URL relativi, sia in termini di href e di targe ("_blank" apre una nuova finestra).
<head> <base href="https://www..." target="_blank" /> </head>

Link

Usato per la creare relazioni, tra il documento e altri documenti o risorse. Usato soprattutto per le relazioni con il CSS. Nel caso di href uguale a #otherId allora andra al tag con quell'Id.

<head> <link rel="stylesheet" type="text/css" href="file.css"/> </head>

Style

Permette di includere stili all'interno del documento. (Meglio di no!)

<style> h1 {color:red;} p {color:blue;} </style>
