Header

Definisce l'intestazione di un documento o di una sezione o di un articolo.

<header> </header>

Nav

Definisce un insieme di link di navigazione (menu o toobla o altri set di link).

<nav> </nav>

Article

Definisce informazioni indipendenti e auto-contenute, cioè è un elemento con un suo senso proprio (indipendente).

<article> </article>

Section

Definisce una sezione del documento.

<section> </section>

Aside

Definisce un contenuto a latere rispetto a quelli principali ma comunque correlati. Puo essere utilizzato per contenere duna barra laterale (citazioni o banne pubblicitari).

<aside> </aside>

Figure

Specifica il contenuto autonomo, come illustrazioni, diagrammi, foto, elenchi di codici, ecc.

<figure> </figure>

Figcaption

Definisce l'elemento di didascalia per il tag figure.

<figure> </figure>

Footer

Definisce un piè di pagina per un documento o una sezione o article. In genere contiene informazioni sulla paternità, copyright, contatti, mappa, torna su e documenti correlati.

<footer> </footer>
